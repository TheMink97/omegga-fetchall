# omegga-fetchall
Fetches all players in your server to you or to a player you specify. 

## Usage
Syntax: `/fetchall [optional: playername]`
Example: `/fetchall cake` (Teleports all players in the server to cake)
If no playername is provided, all players will be teleported to the player that used the command. Furthermore, the command can only be used by the host or by a player that has the admin role.
